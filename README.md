legend:
d = display
bp = breakpoint
br = border
m = margin
mx = margin horizontal (left and right)
my = margin vertical (top and bottom)
p = padding (eg. spacing padding) or percent (eg. border-radius)
px = padding horizontal (left and right)
py = padding vertical (top and bottom)
...
