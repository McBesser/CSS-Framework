CSS-Framework: spacing

cheat sheet

padding and margin
0 to 100 in 4 steps and in 5 steps
eg. 4, 8, 12, 16... 5, 10, 15, 20

margin the same with negative numbers eg. -4, -5 (eg. m--5, bp-1200-mx--5)

padding

  • pl-? = padding: left
  • pr-? = padding: right
  • pt-? = padding: top
  • pb-? = padding: bottom
  • px-? = padding: left, right
  • py-? = padding: top, bottom
  • p-? = padding: left, right, top, bottom

margin

  • ml-? = margin: left
  • mr-? = margin: right
  • mt-? = margin: top
  • mb-? = margin: bottom
  • mx-? = margin: left, right
  • my-? = margin: top, bottom
  • m-? = margin: left, right, top, bottom

breakpoints

eg. bp-1200-m-15

  • ? = eg. p-15 = media: all
  • bp-0-? = eg. bp-0-p-15 = min-width: 0px
  • bp-576-? = eg. bp-576-p-15 = min-width: 576px
  • bp-768-? = eg. bp-768-p-15 = min-width: 768px
  • bp-992-? = eg. bp-992-p-15 = min-width: 992px
  • bp-1200-? = eg. bp-1200-p-15 = min-width: 1200px
