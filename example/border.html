CSS-Framework: border

cheat sheet

border width (0 to 10)

  • br-width-? = border: left, right, top, bottom
  • br-l-width-? = border width: left
  • br-r-width-? = border width: right
  • br-t-width-? = border width: top
  • br-b-width-? = border width: bottom

border style

you can use: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset, initial, inherit

  • br-style-? = border style (left, right, top, bottom)
  • br-l-style-? = border left style
  • br-r-style-? = border right style
  • br-t-style-?: border top style
  • br-b-style-?: border bottom style

border radius

Percent: 0 to 100 in 5th Steps, eg. 5p, 10p, 95p
Pixel: 0 to 50 in 4 Steps (eg. 0, 4, 8...); 0 to 50 in 5th Steps (eg. 5, 10, 15...); and 1 to 10

  • br-radius-? = top left, top right, bottom left, bottom right
  • br-radius-tl-? = top left
  • br-radius-tr-? = top right
  • br-radius-bl-? = bottom left
  • br-radius-br-? = bottom right
  • br-radius-l-? = top left, bottom left
  • br-radius-r-? = top right, bottom right
  • br-radius-t-? = top left, top right
  • br-radius-b-? = bottom left, bottom right

breakpoints

eg. bp-1200-br-radius-15

  • ? = eg. br-radius-15 = media: all
  • bp-0-? = eg. bp-0-br-radius-15 = min-width: 0px
  • bp-576-? = eg. bp-576-br-radius-15 = min-width: 576px
  • bp-768-? = eg. bp-768-br-radius-15 = min-width: 768px
  • bp-992-? = eg. bp-992-br-radius-15 = min-width: 992px
  • bp-1200-? = eg. bp-1200-br-radius-15 = min-width: 1200px
