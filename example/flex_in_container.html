CSS-Framework: flex in container

cheat sheet

example for usefully use flex in container

  1. you need body padding biger or equal item gap eg. body have class px-30 for 30px left and right
  2. you need a container eg. a div with class container-1200 for max width 1200px. If your device width smaler then 1200px is have you 30px spacing.
  3. you need a row container eg. div with class row or flex it's the same
  4. woud you like a gap so add a negative margin to row container eg. gap is 20px so use m--10 for -10px margin
  5. and now add cols with eg. col-25 for 25% width (4 cols) and add gap p-10 for 10px padding (not margin!) summary gap is 20px
  6. done, you cann now add content.
